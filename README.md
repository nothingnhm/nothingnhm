## 🧩 Featured Projects

### 🔵 AD Sentinel: Splunk-Based Active Directory Detection Lab

Built a security monitoring lab with 1 Domain Controller, 2 Windows endpoints, a Kali Linux attacker machine, Splunk SIEM, and a Splunk Universal Forwarder.

**Detection scenarios:** Brute-force login attempts, LLMNR poisoning, Kerberoasting, credential dumping, lateral movement, suspicious PowerShell activity, privilege abuse.

Mapped detection scenarios to **MITRE ATT&CK** techniques and built Splunk dashboards for investigation.

---

### 🔴 Web Application Security Practice Labs

Practicing web application security using authorized labs: DVWA, OWASP Juice Shop, Burp Suite, Kali Linux, Nmap, and browser developer tools.

**Topics covered:** SQL Injection basics, Cross-Site Scripting basics, authentication issues, broken access control, security misconfiguration, information disclosure, directory traversal, parameter tampering, basic vulnerability reporting.

---

### 🔵 SOC Ticket Investigation Portfolio

Documented SOC-style investigations across firewall, IPS, WAF, proxy, VPN, email security, Linux security, cloud security, and endpoint detection alerts.

**Each ticket includes:** alert summary, investigation steps, SIEM/log evidence, IOC identification, impact analysis, severity validation, MITRE ATT&CK mapping, recommended remediation, and final conclusion.

---

### 🔵 Phishing Email Analysis and Incident Response Case Study

Performed phishing investigation using email header analysis, sender verification, URL reputation checks, attachment and hash review, SPF/DKIM/DMARC validation, fake login page analysis, social engineering indicator review, and incident documentation with remediation recommendations.
